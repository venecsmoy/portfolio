# Logo Files

This directory contains logo files used in the portfolio.

## Current Logos

- `wellsky-logo.svg` - WellSky company logo (placeholder - replace with official logo)
- `spotify-logo.svg` - Spotify-style logo for the visualizer project

## Replacing Logos

To use official company logos:

1. Download the official logo files (SVG or PNG format preferred)
2. Rename them to match the existing filenames or update the HTML references
3. Ensure logos are appropriately sized (32x32px recommended)
4. For best quality, use SVG format when possible

## Logo Guidelines

- Keep logos at 32x32px for consistency
- Use transparent backgrounds when possible
- Ensure logos are properly licensed for use in portfolios
- SVG format is preferred for scalability

## Getting Official Logos

- **WellSky:** Request from your manager or marketing team
- **Spotify:** Use official Spotify brand assets from https://developer.spotify.com/branding-guidelines
